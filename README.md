✈️ Flight Data Analysis Dashboard (Power BI)

📌 Overview

This project showcases an interactive flight performance dashboard built using **Microsoft Power BI**.
The goal was to analyze large-scale flight data and extract actionable insights related to delays, cancellations, and operational efficiency.

---

🎯 Objective

* Analyze airline performance using real-world data
* Identify key factors affecting delays and cancellations
* Build a clean, interactive dashboard for decision-making

⚙️ Steps Performed

1. Data Preparation

* Cleaned and transformed raw flight data
* Handled missing and inconsistent values

2. Data Modeling

* Built relationships between tables using Power BI Model View

3. DAX Measures

* Total Flights
* Delayed Flights
* Cancelled Flights
* On-Time Flights
* % Delayed
* % Cancelled
* % On-Time

4. Dashboard Development

* Designed KPI cards for quick insights
* Created bar charts, pie charts, and trend visuals
* Focused on readability and business storytelling

---

📈 Key Insights

* ✈️ Total flights analyzed were **~1.95 million**
* ⏱️ Only **~58% flights were on time**, indicating moderate efficiency
* ⚠️ **~40% flights were delayed**, making delays the primary issue
* ❌ **Cancellations were low (~1.47%)**, showing operational stability
* 🌆 **Atlanta, Chicago, and Dallas** handled the highest flight volumes
* 🛫 **United (UA), Southwest (WN), and Spirit (NK)** showed higher delay percentages
* 🌦️ **Weather was the leading cause of cancellations**
* 📅 Certain days showed **higher cancellation rates**

🧠 Business Impact

* Identified **delays as the primary bottleneck** in flight operations
* Highlighted **high-traffic cities** where optimization can have maximum impact
* Revealed **airline-specific inefficiencies**, enabling targeted improvements
* Showed **weather as a key external risk factor** affecting cancellations
